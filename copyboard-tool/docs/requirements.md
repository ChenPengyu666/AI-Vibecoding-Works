# 需求规格说明书

## 项目概述

**ClipboardHistory** — 一款 macOS 本地剪贴板历史管理工具。自动记录用户复制的文字和图片，提供浏览、搜索、置顶、删除功能，支持自定义存储期限。

## 目标用户

- macOS 普通用户
- 需要频繁回溯复制历史的用户（开发者、编辑、设计师等）

## 功能需求

### F1 - 剪贴板自动记录

- 系统在后台自动监听剪贴板变化
- 捕获文字内容和图片内容
- 连续复制相同内容时不重复记录
- 应用未运行时，启动后从启动时刻开始记录（无法回溯未运行期间的剪贴板历史）

### F2 - 历史浏览

- 主面板以卡片列表形式展示所有历史记录
- 按时间倒序排列（最新在最上面）
- 置顶条目始终排在最前面
- 文字卡片：显示最多 3 行文字预览，超出省略
- 图片卡片：显示缩略图和文件大小

### F3 - 搜索

- 搜索框实时过滤文字条目
- 不区分大小写
- 匹配文字卡片的内容
- 无匹配时显示"没有找到匹配的记录"

### F4 - 置顶与删除

- 每条记录可置顶/取消置顶
- 置顶条目显示蓝色左边框标记
- 每条记录可删除
- 删除操作即时生效，不可撤销

### F5 - 一键粘贴

- 点击文字卡片：将文字内容写回系统剪贴板
- 点击图片卡片：将图片写回系统剪贴板
- 复制成功后卡片有短暂高亮反馈

### F6 - 图片预览

- 点击图片卡片的缩略图区域，弹出大图预览
- 预览窗口支持关闭（点击背景或关闭按钮）

### F7 - 存储期限设置

- 选项：1 天 / 3 天 / 5 天
- 超过期限的记录自动删除
- 置顶条目不受期限影响（不会被自动清理）
- 清理时机：应用启动时 + 每次新增记录时

### F8 - 运行模式

- 模式 1：仅菜单栏 — 菜单栏小图标，点击弹出面板
- 模式 2：仅 Dock — 独立窗口，Dock 有图标
- 模式 3：两者都显示
- 通过设置面板切换，实时生效

### F9 - 开机自启动

- 可开关的开机自启动功能
- 使用 macOS 原生 SMAppService 实现

## 非功能需求

### NF1 - 性能
- 剪贴板轮询间隔 0.5 秒，CPU 占用极低
- 列表支持虚拟滚动，处理数千条记录不卡顿

### NF2 - 隐私
- 所有数据仅存储在本地 `~/Library/Application Support/ClipboardHistory/`
- 不上传任何数据到网络
- 不使用任何第三方分析/统计 SDK

### NF3 - 资源占用
- 内存占用 < 50MB
- 图片存储自动压缩为 PNG 格式

### NF4 - 兼容性
- 最低支持 macOS 14 (Sonoma)
- 同时支持 Intel 和 Apple Silicon Mac
