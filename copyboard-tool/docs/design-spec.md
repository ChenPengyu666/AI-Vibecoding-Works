# UI 设计规范

## 设计原则

- **简洁**：减少视觉噪音，一个操作一个意图
- **直观**：不需要学习，看到就能用
- **一致**：遵循 macOS 原生设计语言

## 配色方案

| 用途 | 色值 | 名称 | 使用场景 |
|------|------|------|---------|
| 主色 | `#7EC8E3` | 天空蓝 | 卡片边框、主按钮、高亮状态 |
| 主色浅 | `#A8D8EA` | 淡蓝 | 卡片悬停背景、次要高亮 |
| 背景白 | `#FFFFFF` | 白色 | 主背景，卡片背景（默认） |

| 文字主 | `#333333` | 深灰 | 正文、卡片标题 |
| 文字辅 | `#666666` | 中灰 | 时间戳、文件大小标注 |
| 边框 | `#E0E0E0` | 浅灰边框 | 卡片默认边框 |

## 卡片设计

### 文字卡片（默认）
```
┌───────────────────────────────────┐
│ 这是复制文字内容，最多显示三行... │  ← 正文(13pt, #333), max 3行
│                                   │
│ 2 小时前              📌  🗑    │  ← 辅助(11pt, #666)
└───────────────────────────────────┘
```
- 边框：1px `#E0E0E0`，圆角 8pt
- 悬停：边框变 `#7EC8E3`，背景微蓝
- 点击：短暂高亮（0.3s 背景变 `#A8D8EA`）

### 图片卡片
```
┌───────────────────────────────────┐
│ [🖼 48×48]  图片 · 128 KB        │
│  缩略图    1 天前      📌  🗑    │
└───────────────────────────────────┘
```
- 左侧缩略图：48×48pt，圆角 4pt
- 右侧文字区：类型标签 + 大小 + 时间

### 置顶卡片额外样式
```
┃ ← 3pt 蓝色竖线 (#5B9BD5)
│
┃  置顶文字内容...
│  2 小时前              📌  🗑
```
- 左侧 3pt `#5B9BD5` 实线
- 背景 `#F5F7FA`

## 图标规范

- 系统 SF Symbols 优先
- 搜索：`magnifyingglass`
- 置顶：`pin.fill`（已置顶）/ `pin`（未置顶）
- 删除：`trash`
- 设置：`gearshape`
- 菜单栏图标：自定义 `clipboard` 风格图标，淡蓝色调

## 空状态

当没有历史记录时：
```
┌───────────────────────────────────┐
│                                   │
│          📋（大图标）             │
│       暂无剪贴板历史              │
│    试试复制一些内容吧 ⌘+C        │
│                                   │
└───────────────────────────────────┘
```

搜索无结果时：
```
┌───────────────────────────────────┐
│                                   │
│      没有找到匹配的记录            │
│                                   │
└───────────────────────────────────┘
```
